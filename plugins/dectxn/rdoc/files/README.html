DecTxn

DecTxn is a Ruby on Rails plugin that provides declarative transaction management.

DecTxn has a dependency on the aquarium AOP framework. 

        sudo gem install aquarium

Installing the CURRENT release of DecTxn. 

        script/plugin install http://dectxn.rubyforge.org/svn/tags/CURRENT/plugins/dectxn

Installing CURRENT with -x (svn:exernals). 

        script/plugin install -x http://dectxn.rubyforge.org/svn/tags/CURRENT/plugins/dectxn

The plugin will install my_transaction_config.rb into the Rails config/initializers dir. Transaction configuration can be added to this file.

Example

The following line specifies calls to all methods in Account that begin "my_" will be handled with required transactional semantics. 

        Txn::required :for_type => Account, :calls_to => /^my_/
